## Pain Point
快递短信多且杂，翻找短信费时间，容易忘取。手动输入取件码繁琐且易出错。

## Core Features
- **智能短信识别**：粘贴一段或多段复杂短信，AI 自动提取取件码、地点和日期。
- **待办清单管理**：将取件信息转化为可视化待办卡片，取完即勾选，清晰明了。
- **本地持久化**：所有取件记录和 API 配置全部保存在浏览器本地，保护隐私且离线可用。
- **跨平台支持**：支持 Web、Windows 桌面和 Android 安卓。

Web / Windows / Android
